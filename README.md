# DeskMate-Frontend

DeskMate is a project created by a group of students (See in a section below), that is about control of desks.
This meaning adjusting height, scheduling positions, seeing error logs, and so much more.

## Requirements

- nodejs

## Installation and Running DeskMate

1. Navigate to the folder that contains the DeskMate folder.

2. Open the folder.

3. Open cmd or Powershell.

4. Run the following commands to install dependencies and run the app (frontend):

```bash
npm install && npm run app
```

5. Open a browser on the given url. By default this would be:

```bash
http://localhost:5173/
```
